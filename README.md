# Recommender
> Application for recommending movies

## Technologies
* PHP - version 8.1.25
* Symfony - version 6.3.8
* PostgreSQL - version 13.2-1

## Local Setup
```
docker compose up -d
```
## Tests
```
docker compose exec php /var/www/recommender/vendor/bin/phpunit
```
